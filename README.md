# 🎯 LeadLens

**Ассистент квалификации лидов для отдела продаж**

---

## О проекте

LeadLens — это минималистичное веб-приложение, которое помогает менеджерам по продажам квалифицировать лидов за 30 секунд. Вместо ручного заполнения CRM и переключения между системами — единый интерфейс с авто-скорингом и готовыми скриптами для звонков.

Проект разработан в рамках хакатона CodeGryphon (трек «Продажи»).

---

## Решаемые боли

| Боль | Решение LeadLens |
|------|------------------|
| Ручная квалификация лидов | Авто-скоринг + чекбоксы для быстрой проверки |
| Нет сквозной аналитики | Экспорт в CSV со всеми метриками для анализа |
| «Зоопарк» систем | Единый интерфейс: скоринг, скрипты, экспорт |
| Нет готовых скриптов | Генерация персонализированных скриптов на основе зрелости лида |

---

## Ключевые возможности

### Двойная система скоринга

| Тип | Как считается | Макс. балл |
|-----|---------------|------------|
| **Потенциал** | Авто из данных (бюджет, timeline, digital presence) | 100 |
| **Квалификация** | Ручная проверка (4 критерия после звонка) | 100 |
| **Приоритет** | Среднее арифметическое двух scores | 100 |

### Авто-приоритизация

- **Высокий приоритет** (≥70) — звонить в течение 24 часов 🔥
- **Средний приоритет** (40-69) — работать в текущей неделе ⚡
- **Низкий приоритет** (<40) — отложить или отклонить 💤

### 💬 Генерация скриптов

На основе **цифровой зрелости** лида (сайт + соцсети + активность):

| Зрелость | Скрипт |
|----------|--------|
| Высокая | Техническое обсуждение, интеграция |
| Средняя | Демонстрация ценности, кейсы |
| Низкая | Вводный звонок, образование |

### Экспорт данных

CSV-файл со всеми метриками:
- Имя, компания, источник
- Бюджет, timeline, digital presence
- Потенциал, квалификация, приоритет
- Статус и все критерии квалификации

### Тёмная/светлая тема

- Переключатель в header (солнце/луна)
- Сохранение выбора в localStorage
- Плавные переходы (0.3s)
- Полная читаемость в обоих режимах

## 🛠 Технологии

| Компонент | Технология |
|-----------|------------|
| Frontend | HTML5 + CSS3 + Vanilla JS |
| Стилизация | CSS Variables (PANTONE Cloud Dancer) |
| Хранение | localStorage (тема, критерии лидов) |
| Экспорт | Blob + CSV (BOM для Excel) |
| Зависимости | 0 (single file, no build) |
